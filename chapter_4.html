学习编程

函数调用

函数是对一堆的封装，编写程序很有可能就会遇到一些基本普偏使用的功能，比如打印字符串，这是C语言标准库里就实现好了的函数，直接引入相关头文件再调用就好；还有一些对一些程序来说没那么基本的功能，这就需要自己来实现了。

函数的使用自然也可以减少程序源代码的行数，增加程序的可读性，也可以让使用者不必在乎函数功能内部的实现逻辑直接拿来就用。

创建函数

创建函数的基本格式是

<return value type> <function name> (arguments ...){
    <code>
}

就像这样：

int add(int a, int b){
    return a + n;
}

这段代码就是创建了一个add()函数用于加两个数。

C语言中，如要调用一个函数，如果调用函数在被调用函数之前，虽然还是可以调用的，编译器会报一个Warnning。调用函数中有过有一个变量接受调用函数的返回值，如果类型不对的话，编译器会报error（至少GCC编译器是这样）。但还是推荐先在源代码的前面写上函数声明，这样一个Warnning都没有了。以add()函数为例，函数声明这么写：

int add(int a, int b);

函数声明就是告诉编译器有这么一个函数，不需要具体的函数实现。

比如下面这个有函数声明的例子：

#include<stdio.h>

int add(int a, int b);

int main(void){
    int a = 1, b = 2;
    printf("a + b = %d", add(a, b));
    return 0;
}
int add(int a, int b){
    return a + b;
}

返回值的数据类型就是在第二章中介绍的数据类型了。

在此顺便介绍一下main()函数的参数：argc和argv。这两个参数的意思是传递给程序的参数的个数和实际参数的值。

如果你没有使用过在终端执行程序，也许很难理解双击执行程序的时候怎么还会有传递参数这种行为。如果是这样的话，我推荐你试一试，不需要GNU/Linux系统，Windows 11自带的Windows Terminal就不错，Win10的话不自带就自己下载吧，你可以在其中执行几个命令试一试。如果你没调整过默认起的话就是是PowerShell，这里介绍几个命令：

下面来一个获取命令行参数的代码：

#include<stdio.h>

int main(int argc, char* argv[]){
    for(int i = 0; i < argc; i++)
        printf("argv[%i]: %s", i, argv[i]);
    return 0;
}
