细🔒

数组在定义的时候需要说明它的元素个数，若是空着不写无法通过编译器的检查，不过如果编译器版本太低的话倒是可以。不过可以使用malloc()动态分配数组，在内存分配那一节将会说这个。

C语言没有字符串的概念，如果要实现字符串的话需要使用字符数组。但是数组需要明确说明元素个数，太麻烦了。这里可以使用指针，就像这样：

char* str = "test";
char* str = "测试";

C语言字符数组对非ASCII字符使用UTF-8编码，汉语在这里占三个字节。可以使用string.h里的strlen()函数证明这一点。

正因为数组和指针内部差不多，你在函数传参中传递一个数组时，比如这样：

// a is a array

int len = strlen(a);

这样传递过去的不过是首位元素的地址，也就是个指针罢了。
