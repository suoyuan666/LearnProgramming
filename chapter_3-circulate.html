学习编程

循环

循环语法总共有三种实现——while循环，do-while循环和for循环。其中最常用的应该是for循环了。

for

一个典型的for循环如下：

for(int i = 0; i < 5; i++){
    <code>
}

上一段代码的含义是，先int i = 0，如果i < 5，就执行内部的code，执行完后再执行i++，然后再做判断。

for循环括号内的三个语句用;分隔，int i = 0和i++这俩部分可以省略，但是分号需要保留。

如果for循环内部的code只有一行的话，也可以像if那样不写{}。

for(int i = 0; i < 5; i++)
    <code>

while

while循环没有for循环这么花哨：

while(<condition>){
    <code>
}

while循环也可以在code只有一行的时候去掉{}。

do-while

do-while循环就是在while判断之前先执行一次。

do{
    <code>
}while(<condition>)

循环的底层表示没有和判断差多少，while和for循环有两种大致格式，这里不介绍do-while循环了，懒得说了。

示例代码（第一种）：

long fact_while(long n)
{
	long result = 1;
	while(n > 1){
		result *= n;
		n = n - 1;
	}
	return result;
}

;n in %rdi
fact_while:
	movl $1, %eax
	jmp .L5
.L6:
	imulq %rdi, %rax;
	subq $1, %rdi
.L5:
	cmpq $1, %rdi
	jg .L6
	rep; ret

示例代码（第二种）：

long fact_while(long n)
{
	long result = 1;
	while(n > 1){
		result *= n;
		n = n - 1;
	}
	return result;
}

;n in %rdi
fact_while:
	cmpq $1, %rdi
	jle .L7
	movl $1, %eax
.L6:
	imulq %rdi, %rax
	subq $1, %rdi
	cmpq $1, %rdi
	jne .L6
	rep; ret
.L7:
	movl $1, %eax
	ret
