在此介绍一下main()函数的参数：argc和argv。这两个参数的意思是传递给程序的参数的个数和实际参数的值。

如果你没有使用过在终端执行程序，也许很难理解双击执行程序的时候怎么还会有传递参数这种行为。如果是这样的话，我推荐你试一试，不需要GNU/Linux系统，Windows 11自带的Windows Terminal就不错，Win10的话不自带就自己下载吧，你可以在其中执行几个命令试一试。如果你没调整过默认起的话就是是PowerShell，这里介绍几个命令：

ls #列出当前目录的文件，如果ls后面空格接文件夹的话就列出那个文件夹的文件
cd #切换目录

这里只介绍了两个命令，我懒了()。

如果想要更多的了解，可以访问计算机教育中缺失的一课

下面来一个获取命令行参数的代码：

#include<stdio.h>

int main(int argc, char* argv[]){
    for(int i = 0; i < argc; i++)
        printf("argv[%i]: %s", i, argv[i]);
    return 0;
}
