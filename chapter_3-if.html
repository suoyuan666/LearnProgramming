学习编程

判断

C语言中，实现判断有两种方法(if和switch)

if

基本格式就是这样：

if (/* condition */)
{
    /* code */
}

这段代码来自于Visual Studio Code的代码补全。括号内部就是判断的命令，比如a < 1这样；也可以是布尔值，比如如果要a[0]这个值不为0才进入这个循环，就可以直接填a[0]就行。

这里的赋值有一些二目运算符，像+=这样的运算符表示的意思就是先加再赋值，比如a += 1，就是a = a + 1。

一元操作符中还有++和--这两个运算符，这俩运算符分别叫做自增和自减运算符。该运算符再变量前还是后的具体含义并不一样：a++和++a的含义并不相同，运算符在前面的会先进行相加，如果运算符再后面则是最后再加。 对于下面的代码：

//filename: jiajia.c
#include<stdio.h>
int main(void){
        int a = 2;
        int b = 3;
        printf("%d\n", b+a++);
        printf("%d\n", ++a);
        return 0;
}

如果是自增自减来回嵌套的话，貌似没有标准规定对这种情况该如何处理，只能看编译器自己如何处理。

如果要对情况不匹配的情况也要单独做一些操作，或者再情况不匹配时再仔细做一波判断的话，就需要使用else关键字了。

if(<condition>)
{
    <code>
}else{
    <code>
}

if(<condition>)
{
    <code>
}else if(<condition>){
    <code>
}

如果if判断成功要进行的操作只有一个的话，不需要大括号也行，就像这样：

if(a > n)
    printf("%d\n", a);

switch

if的判断有一个缺点，如果要进行大量的判断，它会挨个从头判断到成功的地方。switch一定程度上可以避免这个问题，它可以生成一个分支跳转表，可以直接跳转。

switch的基本格式如下：

switch (expression)
{
case /* constant-expression */:
    /* code */
    break;
default:
    break;
}

switch括号里面的是要判断的表达式，case后面（/* constant-expression */这个地方）放表达式的值，code处放具体的执行逻辑。可以有多个case，每个case的code的结尾都要有一个break，否则执行完这个case不会直接跳出switch-case，而是直接向下执行。如果表达式的值没有对应的case，那么就会走default。

这里对比两组代码的汇编代码：

//filename: if.c
#include<stdio.h>
int main(void){
    int a = 5;
    if(a == 1){
        a += 4;
    }else if(a == 2){
        a += 3;
    }else if(a == 3){
        a += 2;
    }
    return 0;
}

//filename: switch.c
#include<stdio.h>
int main(void){
    int a = 5;
    switch(a){
        case 1:
            a += 4;
            break;
        case 2:
            a += 3;
            break;
        case 3:
            a += 2;
            break;
        default:
            a += 1;
            break;
    }
    return 0;
}
