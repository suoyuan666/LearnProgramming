简单数据类型

在上面就已经介绍了两种数据类型了——int和void。

数据类型关键字大小（字节）
字符char1
整数short2
整数int4
整数long8
浮点数float4
浮点数double8

三个整数所占字节大小不同，能表达的整数的范围也就不同，字符是通过ASCII码来表示的，浮点数中，float是单精度，double是双精度。精度可以理解为表达的小数是否精确的精度。float能表达的小数位数是要比double少的。

声明一个变量的基本格式就是<datatype> <varname>

如果在声明一个变量就要给他赋一个初始值的话，还有在后面加上等号还有初始值：

<datatype> <varname> = <data>

int a = 1;
char var = 'a';
double f = 2.5;

char字符需要用单引号包一手，否则会被认为是变量a。=在这里表示赋值，从右往左读。

//filename: datatype.c
#include<stdio.h>
int main(void){
    int a = 1;
    char var = 'a';
    double f = 2.5;
    printf("a: %d\n", a);
    printf("var: %c\n", var);
    printf("f: %f\n", f);
    return 0;
}

~/CodeProject/C/LP
> gcc datatype.c -o datatype

~/CodeProject/C/LP
> ./datatype
a: 1
var: a
f: 2.500000

printf()函数中%开头的转换格式，它以%开始，转换说明符结尾。man-pages中的相关叙述如下

The format string is a character string, beginning and ending in its initial shift state, if any. The format string is composed of zero or more directives: ordinary characters (not %), which are copied un‐changed to the output stream; and conversion specifications, each of which results in fetching zero or more subsequent arguments. Each conversion specification is introduced by the character %, and ends with a conversion specifier. In between there may be (in this order) zero or more flags, an optional minimum field width, an optional precision and an optional length modifier.

The overall syntax of a conversion specification is:

%[$][flags][width][.precision][length modifier]conversion

%d就是整数，%c是字符，%f是浮点数。这里的d、c、f就是转换说明符。更多关于printf()函数的介绍，可以在man-pages中看看。

接下来介绍一手指针，指针是一个存放地址的变量。假设我要定义一个指向int的指针：

int *p;

如果这个指针要指向变量a的话：

p = &a;

如果要访问变量a，需要在p前面加上*解引用，如果直接访问p的话则是访问它的地址。

// filename: point.c
#include<stdio.h>
int main(void){
    int a = 5;
    int *p;
    p = &a;
    printf("a: %d\n", a);
    printf("p: %d\n", p);
    printf("*p: %d\n", *p);
    return 0;
}

~/CodeProject/C/LP
> gcc point.c -o point

~/CodeProject/C/LP
> ./point
a: 5
p: -1384803156
*p: 5

之前介绍的都是变量，代码中还可以定义常量，即不会改变。常量需要使用const修饰变量，比如这样：

const int a = 10;
