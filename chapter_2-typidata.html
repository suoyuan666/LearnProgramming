简单数据类型

在上面就已经介绍了两种数据类型了——int和void。

数据类型关键字大小（字节）
字符char1
整数short2
整数int4
整数long8
浮点数float4
浮点数double8

三个整数所占字节大小不同，能表达的整数的范围也就不同，字符是通过ASCII码来表示的，浮点数中，float是单精度，double是双精度。精度可以理解为表达的小数是否精确的精度。float能表达的小数位数是要比double少的。

声明一个变量的基本格式就是<datatype> <varname>

如果在声明一个变量就要给他赋一个初始值的话，还有在后面加上等号还有初始值：

<datatype> <varname> = <data>

int a = 1;
char var = 'a';
double f = 2.5;

char字符需要用单引号包一手，否则会被认为是变量a。=在这里表示赋值，从右往左读。

//filename: datatype.c
#include<stdio.h>
int main(void){
    int a = 1;
    char var = 'a';
    double f = 2.5;
    printf("a: %d\n", a);
    printf("var: %c\n", var);
    printf("f: %f\n", f);
    return 0;
}

printf()函数中%开头的转换格式，它以%开始，转换说明符结尾。man-pages中的相关叙述如下

The format string is a character string, beginning and ending in its initial shift state, if any. The format string is composed of zero or more directives: ordinary characters (not %), which are copied un‐changed to the output stream; and conversion specifications, each of which results in fetching zero or more subsequent arguments. Each conversion specification is introduced by the character %, and ends with a conversion specifier. In between there may be (in this order) zero or more flags, an optional minimum field width, an optional precision and an optional length modifier.

The overall syntax of a conversion specification is:

%[$][flags][width][.precision][length modifier]conversion

%d就是整数，%c是字符，%f是浮点数。这里的d、c、f就是转换说明符。更多关于printf()函数的介绍，可以在man-pages中看看。

概念介绍：

栈是一个后入先出的数据结构，压入和弹出数据都在栈的一端，称为栈顶，这是寄存器rsp指向的地方，即rsp保存栈顶的地址，rbp指向函数的栈底。下图是在栈的示意图：

从这里可以看出，栈顶在低地址，增大栈的容量就是让rsp保存的值变小。

每个CPU都有一些寄存器，寄存器是一个存储设备，访问速度快但是容量少。32位的CPU有8个寄存器，到64位CPU就有了16个寄存器了。不同的寄存器被规定为有不同的用途，这里不过多介绍了。

前两行是调用函数都有的两行，push用来把数据压入栈中。push %rbp就是要将rbp寄存器的值保存到栈上。mov指令用来把一个地方的值复制到另一个地方，mov %rsp,%rbp就是将rsp寄存器的值复制给rbp寄存器。第一条指令没有生效的时候，rbp保存的还是调用main函数的上一个函数的栈底的地址，push的作用是在将rbp改成main函数栈底之前先将原来的值保存。mov %rsp,%rbp是将当前栈顶的地址给rbp寄存器，此后rbp寄存器不会变，栈顶会一直变化。

sub指令就是减法，sub $0x10,%rsp将rsp的值减了0x10，十进制表示16字节，就是给栈多搞出来了16字节的空间。

第三行的movl是mov指令的变种，用来传递4字节的值。-0xc(%rbp)表达的意思是访问%rbp + -0xc这个地址的值，那么movl $0x1,-0xc(%rbp)这条指令就是把那个地址放上1这个值，之前rsp减去16给函数栈整出来了16字节的空闲空间，rbp-0xc就是指向rsp+4的位置，对应源代码就知道，这个汇编代码应该对应int a = 1;，而int也正是4字节。第四行的 movb $0x61,-0xd(%rbp)中的movb用来传递1字节，这自然就是定义char变量那行代码的汇编代码，0x61是ASCII码中a字符的十六进制表示。

后续的汇编代码就不说了，从1159到116d这几行代码这样的为一组就对应一个printf()函数的调用。

接下来介绍一手指针，指针是一个存放地址的变量。假设我要定义一个指向int的指针：

int *p;

如果这个指针要指向变量a的话：

p = &a;

如果要访问变量a，需要在p前面加上*解引用，如果直接访问p的话则是访问它的地址。

// filename: point.c
#include<stdio.h>
int main(void){
    int a = 5;
    int *p;
    p = &a;
    printf("a: %d\n", a);
    printf("p: %d\n", p);
    printf("*p: %d\n", *p);
    return 0;
}
