数据的表示

引言

在一切开始之前，先以打印输出success的程序为例起一个头

// filename: hello.c
#include<stdio.h>
int main(void){
    printf("success\n");
    return 0;
}

~/CodeProject/C/LP
> gcc hello.c -o hello

~/CodeProject/C/LP
> ./hello
success

编译好后运行该程序，你会发现终端打印了一个单词——success。下面将会单独说一说这几行代码的含义。

第一行以//开头的是注释，注释是给人类阅读的，而非编译器。编译器不会管注释的内容，只会无情的删掉。

第二行以#开头的语句叫做预处理指令，它会在编译程序的整个流程中最开始的时候就被处理。#include表明该程序引入了头文件stdio.h，这个头文件中的一部分代码就描述了第三行的这条语句如何在计算机中实现的，即这条语句如何能够打印这个单词。像打印一些话这样的操作是十分基本常见的，C标准库就实现了很多基本的操作，程序员只需要引入标准头文件就可以使用已经实现好的一些功能（如上面那个例子的打印success）。

第三行定义了一个函数。函数这个概念和数学中的函数类似，有一个x，也相应的有个y。其中，int描述了y的类型（整数，小数还是字符啥的），main是这个函数的名称，括号内部的就是x的类型了，void表明没有x，而且这里函数的x可以不止有一个，可以有多个x，但是只有一个y。main这个名称也不是瞎起的，main在规定中就是程序的入口，虽然这个说法并不严谨，但可以先这么认为。一对大括号圈了一个代码块作为函数的主体，这里面就是main函数所要做的事了。有的语言可以没有入口函数，比如Python，直接从第一行开始走就行。

第四行printf就是一个函数，它接受一个x（就是success\n），但是\n并没有打印出来，这叫做转义字符，\n表示的是换行。传递过去的x通过括号包上，分号表示一个语句的结束，所以它不是将一行看作一条语句，将多条语句写在一行里也是可以的。有的语言并不是这样，比如Python没有分号，一行就表示一条语句。如果你使用Linux系统，那么在终端输入man 3 printf命令可以看到printf()函数更多的信息，如果是Windows的话，man-pages在互联网上可以在线查询，在printf(3) — Linux manual page也可以看到。

第五行return语句表明要返回一个值，return 0表示程序正常退出。int表示返回的y是个整数，这里当然return 0。事实上如果不是int而是void，这个main函数就没必要写return语句了，不过其实这个语句可以省略了，编译器会默认给它整一条这个语句），但是我并不认为这样写法的好处在哪里，如果说是为了省一行return的话，现在int类型的话也可以省这一行了。

常见的数据类型

在介绍各种数据类型之前，先引入对信息的表达进行说明。

信息的表达

无论是什么信息，到底层看都是二进制数。计算机为什么选择二进制在这里我并不准备叙述，这里指说一些相关概念。 1 bit是一个二进制数，8 bit是一个byte，即一个字节。一个十六进制数由4个二进制数表达，一个字节就是两个十六进制数。 在64位的机器上，8字节为一组，32位的机器则是4字节为一组。

简单数据类型

在上面就已经介绍了两种数据类型了——int和void。

数据类型关键字大小（字节）
字符char1
整数short2
整数int4
整数long8

三个整数所占字节大小不同，能表达的整数的范围也就不同，字符是通过ASCII码来表示的，浮点数中，float是单精度，double是双精度。精度可以理解为表达的小数是否精确的精度。float能表达的小数位数是要比double少的。

声明一个变量的基本格式就是<datatype> <varname>

如果在声明一个变量就要给他赋一个初始值的话，还有在后面加上等号还有初始值：

<datatype> <varname> = <data>

int a = 1;
char var = 'a';
double f = 2.5;

char字符需要用单引号包一手，否则会被认为是变量a。=在这里表示赋值，从右往左读。

//filename: datatype.c
#include<stdio.h>
int main(void){
    int a = 1;
    char var = 'a';
    double f = 2.5;
    printf("a: %d\n", a);
    printf("var: %c\n", var);
    printf("f: %f\n", f);
    return 0;
}

> gcc datatype.c -o datatype

~/CodeProject/C/LP
> ./datatype

var: a
f: 2.500000

printf()函数中%开头的转换格式，它以%开始，转换说明符结尾。man-pages中的相关叙述如下

The format string is a character string, beginning and ending in its initial shift state, if any. The format string is composed of zero or more directives: ordinary characters (not %), which are copied un‐changed to the output stream; and conversion specifications, each of which results in fetching zero or more subsequent arguments. Each conversion specification is introduced by the character %, and ends with a conversion specifier. In between there may be (in this order) zero or more flags, an optional minimum field width, an optional precision and an optional length modifier.

The overall syntax of a conversion specification is:

%[$][flags][width][.precision][length modifier]conversion

%d就是整数，%c是字符，%f是浮点数。这里的d、c、f就是转换说明符。更多关于printf()函数的介绍，可以在man-pages中看看。

接下来介绍一手指针，指针是一个存放地址的变量。假设我要定义一个指向int的指针：

int *p;

如果这个指针要指向变量a的话：

p = &a;

如果要访问变量a，需要在p前面加上*解引用，如果直接访问p的话则是访问它的地址。

// filename: point.c
#include<stdio.h>
int main(void){
    int a = 5;
    int *p;
    p = &a;
    printf("a: %d\n", a);
    printf("p: %d\n", p);
    printf("*p: %d\n", *p);
    return 0;
}

> gcc point.c -o point

~/CodeProject/C/LP
> ./point
a: 5
p: -1384803156
*p: 5

之前介绍的都是变量，代码中还可以定义常量，即不会改变。常量需要使用const修饰变量，比如这样：

const int a = 10;

复杂数据类型

这里会介绍数组，结构体，联合和枚举，后两个先立个flag，可惜我现在不太了解他俩，先不说这俩了。

数组

数组即一组数据，这些数据有着统一的数据类型。若要声明一个int类型的数组：

int a[5];

中括号里面的表示这个数组的元素个数，如果你要一一为它们赋初值的话：

int a[3] ={2, 3, 5};

数组是可以嵌套的（数组的元素也是一个数组），也就是所谓的多维数组，这里展示一手二维数组：

int a[2][3] = {{1, 2, 3}, {4, 5, 6}};

typedef int b [3];
b a [2] = {{1, 2, 3}, {4, 5, 6}};

第一行和后两行是等效的。 typedef关键字用于定义一个别名。

访问数组有以下两种方式，以下都以一维数组为例

int a[3] = {1, 2, 3};
printf("%d %d %d", a[0], a[1], a[2]);

这是一种访问方式，a[0]中的0表示数组下标，下标从0开始表示第一个元素。

int a[3] = {1, 2, 3};
printf("%d %d %d", *a, *a+1, *a+2);

上边是指针解引用，数组和指针在底层都是差不多的东西，如果没有数组下标，那么也都是一群地址，直接解引用就可以访问。

结构体

结构体可以将不同的数据类型整到一块。用关键字struct关键字定义。

//filename: struct.c
#include<stdio.h>
int main(void){
    typedef struct a{
        int a;
        float s;
        char n;
    } a;
    a test = {1, 2.0, 's'};
    a te;
    te.a = 1;
    te.s = 5.0; te.n = 'a';
    printf("test.a: %d\n", test.a);
    printf("test.n: %c\n", test.n);
    printf("te.s: %f\n", te.s);
    printf("te.n: %c\n", te.n);
    return 0;
}

~/CodeProject/C/LP
> gcc struct.c -o struct

~/CodeProject/C/LP
> ./struct
test.a: 1
test.n: s
te.s: 5.000000
te.n: a

这里介绍了访问结构体元素的方法，赋值的时候可以像test那样先赋上，或者像te那样都可以。这里用typedef定义别名是为了方便，如果不使用typedef的话，定义一个结构体的元素需要在前面加上一个struct关键字。

结构体有一个内存对齐的问题，内存对齐可以提高访问内存的效率。比如下面的代码：

#include<stdio.h>
int main(void){
    struct a{
        int h;
        double s;
        char n;
    };
    struct b{
        char n;
        int h;
        double s;
    };
    printf("sizeof(a): %d\n", sizeof(struct a));
    printf("sizeof(b): %d\n", sizeof(struct b));
    return 0;
}

~/CodeProject/C/LP --------------------------------------------------------------------------------------------------------
> gcc struct.c -o struct

~/CodeProject/C/LP --------------------------------------------------------------------------------------------------------
> ./struct
sizeof(a): 24
sizeof(b): 16

可以看出，内存占用是不一致的。sizeof()可以计算给定元素的内存占用，不一定偏得是变量啥的，还可以是那些数据类型，比如sizeof(int)
